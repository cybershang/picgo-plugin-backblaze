# 贡献指南

感谢您对 picgo-plugin-backblaze 项目的关注！我们欢迎任何形式的贡献。

## 如何贡献

### 报告问题

如果您发现了 bug 或有功能建议，请通过 [GitHub Issues](https://github.com/cybershang/picgo-plugin-b2/issues) 提交。

提交问题时，请尽量包含以下信息：
- 问题的简要描述
- 复现步骤
- 期望行为与实际行为
- 运行环境（PicGo 版本、Node.js 版本等）
- 相关错误日志

### 提交代码

1. Fork 本仓库
2. 创建您的功能分支 (`git checkout -b feature/amazing-feature`)
3. 提交您的更改 (`git commit -m 'Add some amazing feature'`)
4. 推送到分支 (`git push origin feature/amazing-feature`)
5. 创建一个 Pull Request

### 开发环境设置

```bash
# 克隆仓库
git clone https://github.com/cybershang/picgo-plugin-b2.git
cd picgo-plugin-b2

# 本地测试
npm test
```

### 代码规范

- 使用单引号
- 缩进: 2 个空格
- 使用分号
- 添加适当的注释

## 行为准则

请保持友善和尊重，让我们一起建设一个开放和包容的社区。

## 许可证

通过贡献代码，您同意您的贡献将在 MIT 许可证下发布。
